# IoT Series

This repository is a structured IoT learning series focused on understanding core concepts, architectures, and real-world applications. It covers topics like 3-layer, 4-layer, and 5-layer IoT architectures, explaining how devices, networks, and applications work together.

The content is designed to be simple, practical, and beginner-friendly, with clear explanations, diagrams, and interview-focused questions. This repository also reflects a continuous learning journey, with new topics and improvements added over time.
