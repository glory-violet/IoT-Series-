## 🌐 Overview of 3-Layer IoT Architecture

The 3-Layer IoT Architecture is a simple model that explains how IoT systems work by dividing them into three main layers: Perception, Network, and Application. Each layer has a specific role in collecting, transmitting, and presenting data.

### 1. Perception Layer (Sensing Layer)

**Description:**  
The Perception Layer is responsible for collecting data from the real world using sensors and devices.

**Key Functions:**
- Sense environmental data (temperature, motion, light, etc.)
- Convert physical signals into digital data
- Perform basic local processing

**Examples:**
- Temperature sensors  
- Gas sensors  
- Cameras  
- Microcontrollers (Raspberry Pi, Arduino)

---

### 2. Network Layer (Transmission Layer)

**Description:**  
The Network Layer transfers data collected by devices to cloud platforms or servers.

**Key Functions:**
- Transmit data between devices and systems
- Handle communication protocols
- Ensure reliable data transfer

**Examples:**
- Wi-Fi, Bluetooth, GSM  
- Routers and gateways  
- MQTT, HTTP protocols  

---

### 3. Application Layer (User Layer)

**Description:**  
The Application Layer allows users to interact with the system through applications and dashboards.

**Key Functions:**
- Display data to users
- Provide insights and analytics
- Enable monitoring and control

**Examples:**
- Mobile applications  
- Web dashboards  
- Cloud platforms
